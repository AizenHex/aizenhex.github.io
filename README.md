# Made Reeyza Portfolio

Personal portfolio built with Next.js, MDX, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Edit the app from the Next.js source folders:

- `app/` for routes and global styles
- `components/` for UI sections
- `content/` for project, writing, now, and experience content
- `lib/` for content helpers

## Static Export

```bash
npm run build
```

The GitHub Pages workflow exports the site to `out/` and deploys that folder to
the `gh-pages` branch.
